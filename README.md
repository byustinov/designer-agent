# Designer Agent

AI-агент для UX/UI дизайнера — настраивается за один сеанс.

## Установка

1. Скачай [Claude Desktop](https://claude.ai/download)
2. Открой Claude Desktop → перейди в режим **Cowork** → создай и выбери папку с названием проекта
3. Напиши агенту:

```
Установи https://github.com/byustinov/designer-agent
```

Агент задаст несколько вопросов и настроит всё сам.
