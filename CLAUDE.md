# Designer Agent — Setup

Это репозиторий для настройки AI-агента под UX/UI дизайнера.
Работает в Cowork — пользователь выбирает эту папку и открывает чат.

## Первый запуск

Если файл `.claude/memory/domain/project.md` не существует —
немедленно запусти скилл онбординга:

```
.claude/skills/agent-onboarding/SKILL.md
```

Не жди команды от пользователя. Просто начни.

## Как пользователь попадает сюда

Два варианта:

**Вариант А — через Cowork (рекомендуется):**
1. Создать пустую папку на компьютере (в Finder: Cmd+Shift+N)
2. Открыть Cowork, выбрать эту папку
3. Написать: «Установи designer-agent»

Claude клонирует репозиторий в папку и запускает онбординг автоматически.

**Вариант Б — вручную:**
```bash
git clone https://github.com/byustinov/designer-agent
```
Затем открыть папку в Cowork.

## После онбординга

Когда онбординг завершён и файлы созданы — этот CLAUDE.md
больше не нужен. Агент работает через CLAUDE.md в папке проекта пользователя.
