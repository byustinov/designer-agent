# Designer Agent — Setup

Это репозиторий для настройки AI-агента под UX/UI дизайнера.

## Первый запуск

Если файл `.claude/memory/domain/project.md` не существует —
немедленно запусти скилл онбординга:

```
.claude/skills/agent-onboarding/SKILL.md
```

Не жди команды от пользователя. Просто начни.

## После онбординга

Когда онбординг завершён и файлы созданы — этот CLAUDE.md
больше не нужен. Агент работает через CLAUDE.md в папке проекта пользователя.
