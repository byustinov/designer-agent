# Designer Agent

Это репозиторий с единым скиллом для настройки AI-агента под UX/UI дизайнера.

Пользователь не клонирует этот репозиторий. Агент читает скилл напрямую
через WebFetch и разворачивает структуру в папку пользователя.

## Скилл

`.claude/skills/interface-designer-install/SKILL.md` — самодостаточный файл.
Содержит все шаблоны, логику вопросов и развёртывания без внешних зависимостей.

Raw URL для WebFetch:
`https://raw.githubusercontent.com/byustinov/designer-agent/main/.claude/skills/interface-designer-install/SKILL.md`
